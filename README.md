# Kneeling bus Information System Frontend
저상버스 정보 시스템 

<br/>

## 1. 제작 기간
 - 2023.04.26 ~ 2023.06.08

<br/>

## 2. 실행방법
 1. `npm install` 실행하여 node_modules 폴더를 다운
 2. `npm start` 실행

<br/>

## 3. 사용 기술 및 API
 - React
 - styled-components
 - Kakao 지도 API
 - Naver Maps API

<br/>

## 4. 배포 링크
 - https://stupidtalkingpotatoes.github.io/KIS_frontend/

<br/>
